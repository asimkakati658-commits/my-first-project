# Python Calculator

This is my first Python project uploaded to GitHub.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Runs continuously using `while True`
- Exit option for closing the program

## How to Run

1. Install Python.
2. Download `calculator.py`.
3. Open Terminal or Command Prompt.
4. Run:

```bash
python calculator.py
```
